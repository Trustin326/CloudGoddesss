How to use:
1) Upload the two files to any static host (GitHub Pages, Netlify, your site):

2) Embed with an <iframe>:
   <iframe src="cloudgoddess-widget.html" style="width:100%;height:560px;border:0;border-radius:12px"></iframe>
3) (Optional) Pass a JSON architecture to pre-load:
   Append ?data=URL_ENCODED_JSON to the iframe URL.
